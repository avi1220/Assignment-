      BeautifulSoup's find() and findAll() methods can easily filter HTML (XML) files with different attributes in the HTML code to find the desired tag group or individual tags.

      find_ all()- It returns all the matches or in other words it scans the entire document and returns all the results and the return type will be <class 'bs4.element.ResultSet'>
